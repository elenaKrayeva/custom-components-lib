## How to run the app

**Prerequisites**

**Setup**

```bash

```

**Run Storybook (recommended for local development)**

```bash
npm run storybook

```

**Run tests**

```bash
npm test           # single run
npm run test:watch # watch mode
npm run test:cov   # coverage
```

**Lint & format**

```bash
npm run lint
npm run format
```

**Build the library**

```bash
npm run build
```

**Build Storybook (static site)**

```bash
npm run build-storybook
```
